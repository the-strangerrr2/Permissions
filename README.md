# Permissions

نمونه برنامه اندروید برای درس موبایل ۲ با موضوع Permission ها.

## امکانات

- نمایش وضعیت دسترسی‌های runtime
- درخواست یک‌جای همه دسترسی‌های داده‌نشده
- درخواست جداگانه برای هر دسترسی
- نمایش دلیل نیاز به permission در صورت رد شدن درخواست
- دکمه باز کردن صفحه تنظیمات برنامه

## Permission های استفاده‌شده

- `CAMERA`
- `ACCESS_FINE_LOCATION`
- `READ_CONTACTS`
- `POST_NOTIFICATIONS` برای اندروید ۱۳ و بالاتر

## اجرا

پروژه را در Android Studio باز کنید و Run بگیرید.

APK دیباگ بعد از build در مسیر زیر ساخته می‌شود:

```text
app/build/outputs/apk/debug/app-debug.apk
```
